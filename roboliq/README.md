# Roboliq

Roboliq compiles biological protocols specifications down to low-level commands
for robotic lab automation.

# Setup

```{sh}
npm install
```

# Usage

To run a protocol:

```{sh}
node src/main.js ${PROTOCOL}
```

# Development

To run the tests:

```{sh}
npm test
```

To generate the documentation:

```{sh}
npm run jsdoc
```
